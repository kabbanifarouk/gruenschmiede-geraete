# Grünschmiede – Gerätebestand

Echtes Backend für den Gerätebestand: Next.js (Vercel) + Supabase (Datenbank, Anmeldung, Fotos).

## Einrichtung

1. `schema.sql` im SQL-Editor von Supabase ausführen.
2. Diesen Ordner zu GitHub hochladen.
3. Bei Vercel importieren, Umgebungsvariablen setzen (siehe `.env.example` plus `SETUP_SECRET`).
4. Deployen. Die Startseite leitet automatisch zur Einrichtung, solange noch kein Zugang existiert.

Die ausführliche Anleitung bekommst du separat als Nachricht.
